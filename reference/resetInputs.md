# Reset simulation inputs

Function `resetInputs()` allows resetting state variables in `x` to
their defaults.

## Usage

``` r
resetInputs(x)
```

## Arguments

- x:

  An object of class
  [`spwbInput`](https://emf-creaf.github.io/medfate/reference/modelInput.md)
  or
  [`growthInput`](https://emf-creaf.github.io/medfate/reference/modelInput.md).

## Value

Does not return any value. Instead, it modifies input object `x`.
